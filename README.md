## **Aim:**

To demonstrate the ability of text-to-image generation tools to reproduce an existing image by crafting precise prompts. The objective is to identify key visual elements within the reference image and use these details to generate an image as close as possible to the original through iterative prompt refinement.

---

## **Procedure:**

### **1. Analyze the Given Image:**

Carefully examine the target image and identify:

* **Objects / Subjects:** People, animals, structures, landscapes, items
* **Colors:** Dominant tones, contrast patterns, gradients
* **Textures:** Rough, smooth, glossy, matte, grainy
* **Lighting:** Direction, brightness, shadows, reflections
* **Background:** Simple, detailed, blurred, scenic
* **Composition:** Foreground, mid-ground, perspective, symmetry
* **Style:** Realistic, digital art, hyperrealism, watercolor, cinematic

---

### **2. Create the Basic Prompt:**

Write a short description capturing the **main subject** and **overall theme**.
Example: *“A peaceful field with a small house.”*

---

### **3. Refine the Prompt with More Detail:**

Include specifics such as color tone, layout, emotional mood, and detailed features.
Example:
*“A peaceful green field with a small wooden house under soft golden evening light.”*

---

### **4. Identify Style and Artistic Influence:**

Specify whether the style looks like:

* Realistic photography
* Cinematic shot
* Illustrated artwork
* Watercolor painting
* Digital art render

Example:
*“Photorealistic sunset landscape with warm tones.”*

---

### **5. Adjust and Fine-Tune:**

Add or modify features to capture key elements such as:

* Weather
* Texture detail
* Perspective
* Lens effect
  Example:
  *“Shot with wide-angle lens, high dynamic range lighting, detailed clouds.”*

---

### **6. Generate the Image:**

Use a text-to-image generation tool such as:

* **DALL·E**
* **Gemini Ai (Nano Banana)**
* **MidJourney**
  Input the refined prompt to generate the output.

---

### **7. Compare the Generated Image with the Original:**

Evaluate:

* Color similarity
* Positioning of elements
* Lighting and shadows
* Texture and clarity
* Style resemblance

If differences exist, update the prompt and regenerate.

---
